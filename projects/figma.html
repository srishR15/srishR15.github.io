Figma to HTML/CSS Automation

  • Tech Stack: Python, HTML, CSS, LangChain, Prompt Engineering
  • Github Link: Codebase

AI-Powered Design Conversion: Developed a full pipeline to convert Figma designs into production-ready HTML/CSS: fetch raw JSON → clean & normalize UI tree → generate HTML/CSS via OpenAI GPT.

From Classic to AI Approach: Originally built a rule-based Python/Jinja2 renderer for small UI frames; transitioned to an AI-driven method for accurate, scalable conversion of complex Figma files.

Optimized API & Data Handling: Implemented caching and rate-limit management to prevent 429 Too Many Requests from Figma API, improving reliability for repeated or large requests.

Pixel-Perfect Output & Usability Enhancements: Handles background contrast for white frames, preserves edges, and ensures browser-rendered UI matches Figma mockups closely.

Structured Project Workflow: Organized pipeline into modules (figma_api.py, export_ui_tree.py, openai_generate.py) with outputs stored as /ai/outputAI/output.html + styles.css for easy integration and testing.

Robust & Extensible: Supports both small-scale classic rendering and large-scale AI-powered conversion, allowing future expansion to new Figma designs with minimal changes.
